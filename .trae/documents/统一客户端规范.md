# 统一客户端规范（go-adata）

## 设计目标
- 为所有 HTTP 数据源提供一致的“结构体 + 上下文 + 可配置客户端”模式
- 取消包级包装函数，明确调用方需传入 ctx
- 统一方法命名：Daily/Minute/Current/List 等

## 客户端选项
- WithTimeout、WithProxy、WithUserAgent、WithClient（可选 Headers）

## 已实施模块
- sentiment：alist、northflow、hot、margin、lifting、mine
- stock/info：concept、stockindex（包级 resty 客户端）
- stock/market：indexmarket（结构体方法）、stockmarket（聚合入口调用 *Ctx）、capitalflow/conceptmarket/conceptflow（包级 resty 客户端）

## 示例
```go
ctx := context.Background()
a := alist.New()
daily, _ := a.Daily(ctx, "2024-07-12", 0)

idx := indexmarket.NewIndexMarket(indexmarket.WithTimeout(15*time.Second))
rows, _ := idx.GetDailyEast(ctx, "000001", "2024-01-01", 1, 0)
```
